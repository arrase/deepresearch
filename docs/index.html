Autonomous & Auditable Research

A specialized agent that performs deep-dives into complex, open-ended questions.
Powered by local LLMs via Ollama and high-performance browsing with Lightpanda.

Get Started View on GitHub

Key Features

Autonomous

Decomposes complex queries into subqueries and search intents.

The agent plans its own path to find the most relevant information.

Local-First

Optimized for local models like Qwen 2.5 or Llama 3 via Ollama.

No data leaves your machine except for web searches. High privacy and low latency.

Web-Scale

Uses Lightpanda, a high-performance headless browser, to navigate.

Efficiently extracts content from the real web with Docker-based execution.

Auditable

Every claim is backed by atomic evidence, URLs, and direct quotations.

Ensures transparency and reliability in the final generated reports.

Iterative

Evaluates progress, identifies gaps, and performs follow-ups.

Self-correcting research loops to ensure maximum coverage of the topic.

Rich Output

Generates Markdown, professional PDF reports, or Discord alerts.

Multiple export formats to suit your workflow and communication needs.

Getting Started

Requirements

  • Python 3.11+
  • Docker (for Lightpanda browser)
  • Ollama (running with your preferred model)
docker pull lightpanda/browser:nightly
ollama pull qwen2.5:7b

Installation

git clone https://github.com/yourusername/deepresearch.git
cd deepresearch
python -m venv .venv
source .venv/bin/activate
pip install -e .

CLI Usage

Run the research pipeline directly from your terminal.

# Basic research (outputs report.md) deepresearch "What are the latest breakthroughs in solid-state battery technology in 2024?" # Generate a PDF report deepresearch "Compare Lightpanda vs Playwright" --pdf comparative_analysis.pdf # Send findings to Discord deepresearch "The impact of Llama 3 on local AI" --discord

Skill Integration

DeepResearch can be used as a "skill" by AI agents like OpenClaw and Gemini CLI.

# The agent will run this in the background deepresearch "Comprehensive analysis of solid-state battery tech" --discord
